# Google Apps Script Portfolio

Google Apps Script automations built to improve organisational efficiency at BLWCANADA — eliminating manual, recurring tasks across Google Drive, Sheets, and Gmail.

Each script is self-contained with its own setup guide.

---

## Scripts

| Script | What it does | Time saved |
|--------|-------------|------------|
| [Drive File Alert](./drive-file-alert/) | Automatically notifies the media head by email when files are uploaded to Google Drive — eliminating the need to manually check folders and follow up with contributors | — |
| [BLW Combined Report Suite](./ors-report-generator/) | Generates follow-up and first-timer reports for 6 subgroups from Elvanto data — with grading, health bands, PDF export, and email delivery | 2.5 hrs/week → under 5 mins |

---

## Why Apps Script

These scripts run entirely inside Google's infrastructure — no servers, no installs, no third-party dependencies. They use time-based triggers and spreadsheet-bound menus to run automatically or on demand, and stay comfortably within Google's free tier quota.

---

## Setup (general)

Each script has its own README with full setup instructions. The general pattern is:

1. Go to [script.google.com](https://script.google.com) or open **Extensions → Apps Script** from a Google Sheet
2. Paste the script and save
3. Run once to grant permissions
4. Use the menu or trigger to run it
